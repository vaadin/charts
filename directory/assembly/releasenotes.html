Overview of Vaadin Charts

Vaadin Charts is an add-on for Vaadin Framework. It provides advanced charting functionality for any mobile or desktop Vaadin application.

Overview of changes in 4.0

  • Added Vaadin 8 and Java 8 support.
  • Dropped support for Angular2.
  • Dropped support for React.

Java API Review for 4.0

  • Added DataProviderSeries class, that replaces ContainerDataSeries.
  • Methods that used java.util.Date are marked as deprecated, use Java 8 java.time.Instant instead.

Package Contents

The package contains the following files and directories:

/
 Licensing information, and README. 
addon/
 The add-on itself along with its packaged sources and javadocs. 
addon/apidocs
 Extracted, browsable JavaDocs. 
addon/lib
 Third-party libraries needed by Vaadin Charts. 
webcomponent
 The Web Component version of Vaadin Charts.

Third Party Dependencies

All dependencies are included in the package in the addon/lib directory. Vaadin Charts 4 directly depends on the following third-party libraries:

  • Apache commons-io
  • Jackson databind
  • Polymer
  • webcomponents.js

Migration Notes

Vaadin Charts 3 is mostly API compatible with Vaadin Charts 1.x and 2.x. It can thus be used as a drop-in replacement with minor changes for any 1.x or 2.x version. Vaadin Charts 1.x licenses are valid for Vaadin Charts 3.x if you haven't upgraded to 2.x before.

For details about migration, check the Charts 2 to Charts 3 Migration Guide
